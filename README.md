# Python Simple Soap Server

  
License: MIT


### Setup Project

##### Create virtual env:

    $ python -m venv .venv
    $ source .venv/bin/activate
    $ pip install -r requirements.txt

##### Run project :

     $ python server.py
    
      Soap server is ready

##### Test project : 
    For easy test project can use postman collection



> If you import database this step is not needed
